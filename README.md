# Cloud Compute Tutorials

Welcome to the home of Cloud Compute (CC) tutorials, this is the starting point for learning how to use the USACE Cloud Compute ecosystem to build and scale compute.

Start by reading through the [docs](./docs/README.md) and then work through the [hello world]() tutorial to get familiar with the CC ecosystem and capabilities.

> Note: Tutorial documentation in the `/tutorials/ffrd` folder assume familiarity with the Future of Flood Risk Data (FFRD) program and Standard Operating Procedures (SOP). The tutorials within that folder are intended for Federal Employees and contractors working on the FFRD program and are not designed for general audiences. Before using the FFRD tutorials, familiarize yourself with the FFRD SOP and the associated job aids.
